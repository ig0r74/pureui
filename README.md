pureui
======

Lightweight responsive framework

Demo: http://ig0r74.github.io/pureui/

####Features:
* hamburger menu on small screen's
* buttons
* two-level top menu
 
####Added classes:
.hide-sm, .hide-md, .hide-lg, .hide-xl

.visible-sm, .visible-md, .visible-lg, .visible-xl
